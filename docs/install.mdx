## Install iii

Install the iii engine binary via the official install script from `install.iii.dev`.

## Verify installation

Run `iii --version` to confirm the binary is on your PATH and reports the expected version.

## Upgrade iii after install

Use `iii update` to upgrade the engine and its managed binaries — see [CLI → Updating iii itself](/using-iii/cli#updating-iii-itself).

## VS Code extension

Optional editor support for working in iii projects.

## Agent skills

Install the iii skill bundles so coding agents can work with iii in your editor.
