---

iii can be installed in a single step

### 1. Install iii

Install iii and optionally run `iii-cli update` to install supporting applications.

```bash
curl -fsSL https://install.iii.dev/iii/main/install.sh | sh
iii-cli update
```

### 2. Verify installation

Check that iii has installed correctly with the following command. It should return a version number.

```bash
iii --version
```
