## Find a worker

Search the iii registry for workers that provide a capability you need.

## Add a worker from the registry

Install a published worker into your running system with `iii worker add <name>`.

## Pin a version

Choose a specific version of a registry worker rather than tracking the latest.

## Update a worker

Pull a newer version of a registry worker into a running system.

## Remove a worker

Drop a registry worker from the system.
